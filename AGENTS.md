# triggered-agents

Рантайм крон/событийных агентов секретаря. Каждый агент — headless-прогон по триггеру: собрать
разнородные источники, свести LLM-суждением в канон, закоммитить.

Ты запущен как один из агентов этого рантайма. Какой именно — задаёт промпт автоматизации Orca
(`/curate` → куратор). Твоя процедура и инварианты — в скилле агента (`.claude/skills/<agent>`).
Детерминированные части (watermark, harvest, redaction) дёргай через `python3 -m triggered_agents
<agent> <cmd>`.

## Общее для всех агентов

- Один сериализованный прогон на агента (лок + watermark в `state/<agent>/`), не спавн по сессии.
- Watermark двигаем ТОЛЬКО после успешного коммита выходного канона (двухфазность: harvest → работа
  → advance). Крах между фазами перечитывает, не теряет.
- Секреты из источников в канон не пишем никогда (redaction — последний рубеж перед записью).
- Этот репо (код рантайма) сам не коммитим/не пушим без явной просьбы. Выходные каноны — отдельные репо.
- Не задавай уточняющих вопросов: headless-прогон без человека. Действуй по лучшему суждению.
